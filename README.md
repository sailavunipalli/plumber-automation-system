# Plumber Lead Automation System

A full end-to-end automation system built for emergency plumbing businesses.
When a customer submits a lead form, the system automatically notifies the
plumber and follows up with the customer — zero manual work required.

## Live Demo
👉 [View Landing Page](https://sailavuni.github.io/landingpage/)

## How It Works

1. Customer fills form on landing page

2. N8N webhook receives the data
 
3. Lead saved to Google Sheets (CRM)

4. WhatsApp message sent to the customer
   
5. Email notification sent to the plumber

## Tools Used

- **N8N** — Automation workflow engine
- **Tally.so** — Lead capture form
- **GitHub Pages** — Free landing page hosting
- **Google Sheets** — CRM to track all leads
- **WhatsApp Cloud API (Meta)** — Instant customer messaging
- **Gmail** — Plumber email notifications
